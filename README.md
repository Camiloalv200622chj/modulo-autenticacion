# 📌 Módulo de Autenticación - Colegio San José

## 🎯 Objetivo del Proyecto

Desarrollar un sistema de autenticación para el control de asistencia del Colegio San José, permitiendo el acceso seguro de docentes y administrativos, así como la gestión de usuarios y visualización de información relevante.

## 🧠 Historias de Usuario

### HUS-01: Autenticación de Usuarios

Como usuario del sistema
Quiero ingresar mis credenciales
Para acceder de forma segura a la plataforma

---

### HUS-02: Registro de Usuarios

Como administrador
Quiero registrar nuevos usuarios
Para habilitar su acceso al sistema

---

### HUS-03: Visualización de Dashboard

Como docente
Quiero visualizar un tablero de control
Para conocer el estado de asistencia de mis grupos
